\section{Grafos}

Investiga y representa visualmente como grafos cada uno de los siguientes \textbf{símbolos mágicos de Islandia}

\begin{enumerate}
	\item Kaupaloki
	\item Smjörhnútur (considera sólo líneas completas (no cambios de dirección))
	\item Nábrókarstafur
\end{enumerate}

Para cada uno de ellos, menciona si

\begin{itemize}
	\item El grafo es completo
	\item El grafo es bipartito
	\item Cuál es el grado máximo del grafo
	\item El grafo contiene un Tour Euleriano
	\item ¿Existe alguna manera de reducir el grafo a sus componentes? En caso de que sí, demuéstralo gráficamente.
\end{itemize}
